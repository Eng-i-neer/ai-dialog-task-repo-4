# 跨国物流对账说明（供 Agent 读取）

## 1. 文档目的

本文用于帮助 Agent 快速理解两类对账单的含义、常见费用项目、资金流向，以及如何判断某一票订单是否属于 COD、是否为正常收费或账务冲回。

---

## 2. 两份账单分别代表什么

### 2.1 代理给我们的账单
这是代理公司（如物流渠道商）给我们开的账单，体现的是：

- 代理向我们收什么钱
- 代理退什么钱
- 某些异常费用如何冲回或补退

常见 sheet 可能包括：

- COD
- 上架费
- 尾程退件操作费
- 服务费
- DE/GR/HR/IT 地派服务费
- 尾程运费
- 目的地增值税
- 转寄操作费
- 短信费
- 头程运费
- 尾程退件操作费（补退）

### 2.2 我们给顾客的账单
这是我们最终与顾客结算的账单，体现的是：

- 顾客这批订单一共回了多少 COD 款
- 要扣掉多少运费和杂费
- 最终我们应付给顾客多少钱

常见 sheet 可能包括：

- 期 COD 回款
- 期运费
- 期尾程杂费
- 超期仓储
- 理赔

---

## 3. 最核心的结算逻辑

顾客应收回款通常可以理解为：

```text
顾客净回款 = COD 代收货款
           - 运费
           - 杂费
           - 税费
           - 手续费
           ± 补退 / 扣回 / 理赔等调整
```

更口语化地说：

```text
COD - 运费和杂费 = 我们要给顾客的钱
```

但严格来说，还要考虑税费、手续费和异常调整项。

---

## 4. 资金到底是谁收的

日常看到的费用，基本可以分为 4 类。

### 4.1 顾客付给我们的钱
最典型的是 **COD 代收货款**：

- 顾客签收时付款
- 物流/代理代为收款
- 最终结算到我们，再由我们结给顾客

常见字段：

- 代收金额
- 金额(CNY)
- 小计

### 4.2 代理向我们收的钱
这类最多，包括：

- 头程运费
- 尾程运费
- 地派服务费
- 服务费
- 上架费
- 转寄操作费
- 退件操作费
- 短信费
- 增值税
- 偏远费

本质是：代理或物流商在各环节提供服务，所以向我们收费。

### 4.3 我们再向顾客结算的钱
很多客户账单里的费用，本质上来源于代理账单，只是：

- 代理先向我们收费
- 我们再按订单分摊后向客户结算

例如：

- 运费
- 杂费
- 税费
- 仓储费
- 某些特殊附加费

### 4.4 之前收错、现在退回的钱
这类往往表现为负数，包括：

- 签收件，拒收返程费补退
- 尾程退件操作费（补退）
- 扣回代收款
- 实为退件

本质不是新收费，而是：

- 以前先记错
- 现在冲回 / 补退 / 扣回

---

## 5. “代理”到底是什么

在这个业务里，**代理**可以理解为：

> 夹在“我们”和“最终海外派送公司/仓库/COD 收款方”之间的中间物流服务商。

关系可以理解为：

```text
顾客 → 我们 → 代理 → 海外当地派送商 / 仓库 / 清关 / COD 收款方
```

代理的作用通常包括：

- 整合物流线路
- 对接海外仓和当地派送商
- 安排头程与尾程
- 代为汇总费用
- 出具对账单
- 有时还负责代收代付

代理不一定亲自完成所有动作，但负责把整条链路串起来，并统一与我们结算。

---

## 6. 各类费用按物流流程怎么理解

### 6.1 头程运费
**是谁收：** 代理 / 物流商  
**在哪收：** 货从国内发出，到目的国之前  
**本质：** 中国到海外的主运输费用

常见理解：

- 国内揽收
- 干线运输
- 国际运输
- 到德国、意大利等目的国前的运输成本

### 6.2 尾程运费
**是谁收：** 代理 / 目的国派送商  
**在哪收：** 到了目的国后，最后派送给收件人  
**本质：** 海外本地派送费

### 6.3 地派服务费
**是谁收：** 目的国代理 / 当地派送服务商  
**在哪收：** 目的国落地后处理与派送环节  
**本质：** 当地派送服务基础成本

例如：

- DE = 德国
- GR = 希腊
- HR = 克罗地亚
- IT = 意大利

可以粗略记成：

- 地派服务费 = 目的国本地处理服务费
- 尾程运费 = 目的国本地运输/派送费

### 6.4 服务费
**是谁收：** 代理  
**在哪收：** 平台/渠道/整体服务层面  
**本质：** 综合服务费

可能包括：

- 系统服务
- 渠道服务
- 订单处理服务
- 对账/代收/管理服务

### 6.5 COD 手续费 / 代收手续费
**是谁收：** 代理 / 尾程派送商 / 代收货款服务方  
**在哪收：** 收件人签收付款时  
**本质：** 帮忙代收货款的服务费

即：

> 物流帮忙把钱从收件人那里收上来，这个收款动作本身要收费。

### 6.6 上架费
**是谁收：** 仓库 / 代理  
**在哪收：** 货物入仓、登记、上架时  
**本质：** 入库处理费

### 6.7 转寄操作费
**是谁收：** 代理 / 仓库 / 操作方  
**在哪收：** 货先入仓，再改派、再转发时  
**本质：** 二次处理、重新出库、重新安排运输的操作费

### 6.8 尾程退件操作费
**是谁收：** 目的国代理 / 派送商  
**在哪收：** 尾程派送失败后进入退件流程  
**本质：** 处理退件动作的钱

典型场景：

- 收件人拒收
- 派送失败
- 地址错误
- 包裹退回代理仓

### 6.9 拒收返程费
**是谁收：** 代理 / 派送商  
**在哪收：** 包裹被拒收后返回  
**本质：** 从客户地址退回的运输/处理费

### 6.10 退件入仓费
**是谁收：** 海外仓 / 代理  
**在哪收：** 退件回仓后重新入库  
**本质：** 退回仓库后的重新入库费用

三者可这样区分：

- 退件操作费：处理退件流程本身
- 返程费：退回来的运输费用
- 退件入仓费：退回仓后的入库费用

### 6.11 目的地增值税 / 增值税
**是谁收：** 目的国税务体系（通常由代理代垫代收）  
**在哪收：** 清关或目的地交付阶段  
**本质：** 税

### 6.12 偏远费
**是谁收：** 派送商 / 代理  
**在哪收：** 尾程派送  
**本质：** 偏远地区附加费

### 6.13 短信费
**是谁收：** 代理 / 派送渠道  
**在哪收：** 派送通知时  
**本质：** 给收件人发送通知短信的费用

### 6.14 旺季附加费 / 附加费
**是谁收：** 代理 / 渠道商 / 派送商  
**在哪收：** 特殊时期或特殊条件下  
**本质：** 旺季、爆仓、资源紧张导致的附加成本

### 6.15 超期仓储
**是谁收：** 仓库 / 代理  
**在哪收：** 仓储阶段  
**本质：** 货物存放超时产生的仓租

### 6.16 理赔
**是谁出：** 物流方赔付，或我们对客户进行赔付结算  
**场景：** 丢件、破损、异常  
**本质：** 异常损失补偿

## 7. 如何理解“有些订单有 COD，有些没有”

这不一定表示“不同批次的 COD”，更常见的是：

> 同一份账单里，本来就混有 COD 单和非 COD 单。

### 7.1 有 COD 的情况
订单满足以下条件时通常会有 COD：

- 走的是货到付款
- 客户签收时付款
- 渠道支持代收货款

通常会出现字段：

- 代收金额
- 汇率
- 金额(CNY)
- COD手续费
- 小计

### 7.2 没有 COD 的情况
以下情况可能没有 COD：

- 客户预付款，不是货到付款
- 只是普通物流单
- 所选国家/线路不支持 COD
- 订单取消
- 派送失败
- 后续退件
- 原本与 COD 相关，但已被冲回

### 7.3 有些订单只有费用，没有 COD
一票订单即使没有 COD，也可能仍然出现在：

- 运费表
- 杂费表
- 退件表

因为：

> 有没有 COD，和这票有没有运费/杂费，是两回事。

### 7.4 COD 后续被冲回的情况
有些订单本来和 COD 有关，但后续没有形成有效回款，例如：

- 客户拒收
- 实际退件
- 签收异常
- 代收款被扣回

这时账单可能出现：

- COD 相关字段
- 但金额为负数
- 或备注为“实为退件”“扣回代收款”

---

## 8. 如何理解负数、补退、扣回

对账时，不能只看科目名，还必须看：

- 金额是否为负数
- 备注是否出现以下关键词：
  - 补退
  - 冲回
  - 扣回
  - 实为退件
  - 签收件补退

一般来说：

- **正数**：通常表示正常收费或正常回款
- **负数**：通常表示退款、冲销、补退或扣回

例如：

### 8.1 “实为退件，扣回代收款”
意思是：

- 这票之前看起来像有 COD 回款
- 后来发现实际是退件
- 因此之前计入的代收款要被追回
- 所以出现负数冲销

### 8.2 “签收件，拒收返程费补退”
意思是：

- 之前误按拒收退件收了一笔返程费
- 后来确认其实是签收件
- 因此把误收的费用退回来

### 8.3 “尾程退件操作费（补退）”
意思是：

- 之前按退件收了费用
- 后来确认不该收
- 所以补退

## 9. 常见字段的人话解释

### 9.1 在 COD 回款表中

#### 代收金额
客户签收时支付的货款原币金额。

#### 汇率
将原币换算成人民币时使用的汇率。

#### 金额(CNY)
代收货款折算成人民币后的金额。

#### COD 手续费(CNY)
代收货款服务费。

#### 小计
通常可以理解为：

```text
小计 = 金额(CNY) - COD 手续费
```

即这票理论上的净回款。

如果备注写明“实为退件”“扣回代收款”，则该小计可能为负数，表示冲回。

### 9.2 在运费表中

#### 头程计费重量
用于计算头程费用的重量，不一定等于实重。

#### 尾程计费重
用于计算尾程费用的重量。

#### 头程运费
国内到海外的运输费用。

#### 尾程运费
目的国本地派送费用。

#### 代收手续费
COD 收款服务产生的费用。

#### 附加费 / 旺季附加费
特殊时期或特殊条件下的附加费用。

### 9.3 在尾程杂费表中

#### 上架费
货物入仓上架费用。

#### 尾程退件入仓费
退件重新入仓的费用。

#### 二派费
第一次未成功派送后，第二次派送产生的费用。

#### 增值税
目的国税费。

#### 偏远费
偏远地区派送附加费。

#### 小计
这票订单尾程杂费的合计。

#### 备注
说明这笔费用为什么收、为什么退、为什么补差。

## 10. 核对账单时的实用判断方法

看到一条费用，依次问 3 个问题：

### 10.1 这是哪一段的钱？
- 中国段 / 发出前：头程
- 海外派送阶段：尾程 / 地派
- 仓储处理：上架 / 仓储 / 转寄
- 收钱动作：COD 手续费
- 税：增值税
- 异常：退件 / 二派 / 偏远 / 理赔

### 10.2 这是谁收的？
- 税务系统收：税
- 派送商收：尾程、偏远、二派
- 仓库收：上架、仓储、入仓
- 代理收：服务费、操作费、转寄费
- 代收服务方收：COD 手续费

### 10.3 这是正常收费还是账务调整？
重点看：

- 金额正负
- 备注关键词

判断规则：

- 正数 + 无异常备注：通常是正常收费/正常回款
- 负数 + 补退/扣回/实为退件：通常是账务调整

## 12. 两个最容易混淆的点

### 12.1 “代理收我们的钱”和“我们给顾客的钱”不是一回事
- **代理账单**：从代理视角出发，体现代理向我们收费或退款
- **客户账单**：从我们视角出发，体现我们向客户结算应付或应扣

所以同一票货，可能会同时出现在两张表里，但意义不同。

### 12.2 科目名看起来像收费，不代表这次一定在收费
例如：

- 尾程退件操作费
- 拒收返程费
- COD

如果该行：

- 是负数
- 或备注写明补退 / 扣回 / 冲回

那它更可能是账务调整，不是新增收费。

## 13. Agent 处理这类表格时的建议

Agent 在阅读此类账单时，建议按以下顺序理解：

1. 先区分：这是代理给我们的账单，还是我们给客户的账单
2. 再区分：这行是回款、运费、杂费、税费还是异常调整
3. 再看：金额正负
4. 再看：备注关键词
5. 最后再判断：这笔钱是正常收费、正常回款，还是补退/扣回

建议优先提取以下字段：

- 订单号
- 转单号 / 运单号
- 目的地
- 品名
- 代收金额
- 汇率
- 金额(CNY)
- COD手续费
- 小计
- 头程运费
- 尾程运费
- 上架费
- 退件相关费用
- 增值税
- 偏远费
- 备注

---

## 14. 一句话总结

对于这类跨国物流对账单，可以用一句话概括：

> 顾客签收产生 COD 回款，代理在各物流环节向我们收取运费和杂费，我们再把这些成本分摊后与客户结算；若出现退件、拒收、错收等情况，则会通过负数、补退、扣回等方式在账单中做调整。
