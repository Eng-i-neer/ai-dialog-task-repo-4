## 文件详细说明

### `代码文档管理总控.md` — 代码文档管理系统
- **作用**：@ 引用时自动扫描项目中所有代码文件，检查是否有尚未记录在 README.md 中的新文件，并自动补充文档说明
- **核心功能**：扫描文件 → 对比差异 → 分析新文件 → 更新 README → 更新追踪表
- **文件状态追踪表**：记录所有文件的文档状态（已记录 / 新增未记录 / 已删除）

### `自动执行规则.md` — AI 自动执行规则
- **作用**：定义三条自动执行规则，粘贴到 IDE 设置后每次对话自动生效
- **规则 1**：命令行错误自动防范 + 自动记录
- **规则 2**：完成重要任务后自动追加对话记录
- **规则 3**：创建/修改文件后自动更新代码文档
